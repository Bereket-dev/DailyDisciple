# Daily Disciple - Screen Reference Guide

Visual and functional overview of all 8 screens with key elements.

## 1. Onboarding Screen 📖

**Flow**: Landing → 4 Swipeable Cards → Get Started

### Visual Elements
```
┌─────────────────────────────────┐
│                                 │
│   📖 (Illustration Circle)      │
│                                 │
│   Take Notes                    │
│   Capture sermon insights...    │
│                                 │
│   ● ○ ○ ○  (Progress Dots)     │
│                                 │
│        [Next Button] →          │
└─────────────────────────────────┘
```

### Cards
1. 📖 Take Notes
2. 💭 Reflect
3. ⚡ Apply
4. 📈 Grow

### Key Features
- Smooth horizontal scrolling
- Animated dot indicators
- "Get Started" navigates to MainApp
- Emoji-based visual style

## Responsive Design Notes

- All layouts adapt to screen width
- Cards maintain 24px padding
- Text is readable at all sizes
- Touch targets are 44pt minimum
- Safe area respected for notches
