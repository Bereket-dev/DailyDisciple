# Daily Disciple - Design System

## Color Palette

### Primary Colors
- **Primary**: `#2D3E50` - Deep Navy (Trust, Spirituality)
- **Secondary**: `#D4AF37` - Subtle Gold (Inspiration, Growth)
- **Accent**: `#E8D4B8` - Warm Neutral (Calm, Peace)

### Semantic Colors
- **Background**: `#FAFAF8` - Soft White
- **Surface**: `#FFFFFF` - Pure White
- **Text**: `#2D3E50` - Primary Text
- **Text Light**: `#6B7280` - Secondary Text
- **Success**: `#10B981` - Completed Actions
- **Border**: `#E5E7EB` - Subtle Dividers
- **Shadow**: `rgba(45, 62, 80, 0.08)` - Soft Elevation

## Typography

### Font Family
- Primary: Space Grotesk (modern, clean)
- Fallback: System sans-serif

### Type Scale
- **H1**: 32px, 700 weight, 40px line height
- **H2**: 24px, 600 weight, 32px line height
- **H3**: 20px, 600 weight, 28px line height
- **Body**: 16px, 400 weight, 24px line height
- **Body Small**: 14px, 400 weight, 20px line height
- **Caption**: 12px, 500 weight, 16px line height

## Spacing System

- **XS**: 8px
- **SM**: 12px
- **MD**: 16px
- **LG**: 24px
- **XL**: 32px

## Border Radius

- **SM**: 8px - Small components
- **MD**: 12px - Buttons, inputs
- **LG**: 16px - Cards, major components
- **XL**: 24px - Large rounded containers

## Shadows

### Shadow Small
- Offset: 0, 2px
- Opacity: 10%
- Radius: 4px

### Shadow Medium
- Offset: 0, 4px
- Opacity: 12%
- Radius: 8px

### Shadow Large
- Offset: 0, 8px
- Opacity: 15%
- Radius: 12px

## Component Specs

### Buttons
- **Primary**: Navy background, white text, 16px padding
- **Secondary**: Warm neutral background, navy text
- **Sizes**: Small (12px py), Medium (16px py), Large (24px py)

### Cards
- **Padding**: 24px
- **Border Radius**: 16px
- **Shadow**: Medium
- **Background**: White surface

### Bottom Navigation
- **Height**: 64px (with safe area)
- **Emoji Icons**: 20px
- **Active**: Primary color
- **Inactive**: Light text color

## Design Principles

1. **Minimalist**: Clean layouts with generous whitespace
2. **Peaceful**: Soft shadows, rounded corners, warm colors
3. **Accessible**: High contrast, readable fonts, clear hierarchy
4. **Modern**: 2026 design trends, card-based layouts
5. **Spiritual**: Inspiring color combinations, meaningful iconography

## Responsive Design

- Designed for iOS and Android
- Touch-friendly component sizes (minimum 44px)
- Flexible layouts for various screen sizes
- Safe area consideration for notches

## Animation & Transitions

- Tab transitions: 200ms ease-in-out
- Button press: 100ms opacity
- Card entry: 300ms fade-in
- Bottom tab bar: smooth transitions

## Accessibility

- Minimum contrast ratio 4.5:1
- Touch targets at least 44x44pt
- Semantic color usage
- Clear labels and descriptions
- Icon + text combinations
